# ⚡ 快速启动指南

## 🎯 5分钟上手

### 步骤1：安装依赖（首次使用）
```bash
cd D:\code\twoGradeWord
npm install
```
⏱️ 预计需要1-2分钟

### 步骤2：启动应用
```bash
npm run dev
```
⏱️ 几秒钟后会自动打开浏览器

### 步骤3：开始使用
1. 页面自动打开 http://localhost:3000
2. 点击"🎮 开始闯关"
3. 选择"🌱 简单模式"
4. 点击"开始闯关"
5. 开始答题！

## 💡 重要提示

### ✅ 推荐配置
- **浏览器**：Chrome 或 Edge（最新版本）
- **屏幕**：建议使用电脑或平板
- **网络**：无需联网（本地运行）

### ⚠️ 注意事项
1. **不要清除浏览器缓存**：会导致学习数据丢失
2. **定期导出数据**：在"设置"页面导出备份
3. **使用同一浏览器**：数据存储在浏览器本地

### 🎯 学习建议
- **新手**：从简单难度开始
- **练习**：每天坚持2-3关
- **复习**：定期查看错词本
- **目标**：正确率达到85%+

## 🚨 故障排除

### 问题1：npm install 报错
**解决方案：**
```bash
# 清除缓存后重试
npm cache clean --force
npm install
```

### 问题2：端口被占用
**解决方案：**
- 关闭其他占用3000端口的程序
- 或修改 vite.config.js 中的端口号

### 问题3：页面空白
**解决方案：**
1. 按F12打开浏览器控制台
2. 查看报错信息
3. 刷新页面（Ctrl+F5）

### 问题4：词库加载失败
**解决方案：**
- 确认 `words/wordsSource.json` 文件存在
- 检查JSON格式是否正确
