# 📖 使用说明

## 🚀 启动应用

### 1. 安装Node.js环境
确保已安装Node.js（推荐v16+）

### 2. 安装项目依赖
打开命令行，进入项目目录，执行：
```bash
npm install
```

### 3. 启动开发服务器
```bash
npm run dev
```

### 4. 打开浏览器
访问显示的地址（通常是 http://localhost:3000）

---

## 🎮 功能使用指南

### 一、首次使用

1. **打开应用**
   - 首次打开会自动初始化用户数据
   - 显示学习进度卡片（初始为0）
   - 四个功能入口按钮

2. **开始第一次闯关**
   - 点击"🎮 开始闯关"
   - 选择难度（建议从"简单"开始）
   - 选择模式（选择"全词库"）
   - 点击"开始闯关"按钮

### 二、答题流程

#### 题目显示
- **模式A**：显示拼音，选择对应的汉字
  ```
  题目：táng
  选项：塘  糖  汤  堂
  ```

- **模式B**：显示汉字，选择对应的拼音
  ```
  题目：塘
  选项：táng  tāng  tǎng  tàng
  ```

#### 答题操作
1. 阅读题目（拼音或汉字）
2. 点击4个选项中的一个
3. 系统立即判断正误
4. 显示反馈动画
   - ✅ 答对：绿色高亮 + "太棒了！"
   - ❌ 答错：红色高亮 + 显示正确答案
5. 点击"下一题"继续

#### 连对奖励
- 连续答对5题：🔥 +10分
- 连续答对10题：🔥 +30分
- 实时显示连对数

#### 关卡完成
- 完成所有题目后自动跳转到结算页
- 显示本关成绩、得分详情
- 可选择：再来一关 / 查看错词 / 返回主页

### 三、错词本功能

#### 查看错词本
1. 主页点击"📕 错词本"
2. 看到三个统计数字：
   - 总错词数
   - 未掌握数量
   - 已掌握数量

#### 筛选和排序
- **筛选**：全部 / 未掌握 / 已掌握
- **排序**：错误次数↓ / 时间↓

#### 错词信息
每个错词显示：
- 汉字和拼音
- 来源（哪个课文）
- 错误次数
- 最近错误时间

#### 标记掌握
- 点击"标记掌握"按钮
- 确认后词条变为"已掌握"状态

#### 错词复习
1. 点击底部"开始错词复习"按钮
2. 自动进入难度选择页（模式已设为"错词复习"）
3. 选择难度，开始专项训练
4. 连续答对3次自动标记为"已掌握"

#### 清除已掌握
- 点击"清除已掌握"按钮
- 确认后删除所有已掌握的错词

### 四、学习统计

#### 总体统计
- 📝 总题目数：累计答题数量
- ✅ 答对数：累计答对数量
- 🎯 总正确率：整体正确率百分比
- ⏱️ 学习时长：累计学习时间

#### 最近7天学习情况
显示每天的：
- 日期
- 难度模式
- 得分
- 正确率（颜色标识）

#### 分难度统计
🌱 简单 / 🌿 中等 / 🌳 困难
每个难度显示：
- 题目数
- 答对数
- 正确率
- 进度条

#### 最易错TOP5
显示错误次数最多的5个词汇：
- 排名
- 汉字和拼音
- 来源
- 错误次数

### 五、设置功能

#### 用户信息
- 查看用户名
- 查看当前等级
- 查看加入时间

#### 音效设置
- 开启/关闭音效（功能预留）
- 调整音量（功能预留）

#### 动画设置
- 开启/关闭动画（功能预留）

#### 数据管理
1. **导出学习数据**
   - 点击"导出学习数据"按钮
   - 下载JSON格式的备份文件
   - 文件名：学习数据_YYYY-MM-DD.json

2. **清除所有数据**
   - 点击"清除所有数据"按钮
   - 确认后删除所有学习记录
   - ⚠️ 此操作不可恢复！

## 💡 使用技巧

### 学习建议

#### 新手入门
1. 从"简单"难度开始
2. 选择"全词库"模式
3. 先熟悉题目形式（模式A和模式B）
4. 不要着急，仔细看题

1. 逐步提高难度（简单 → 中等 → 困难）

4. 争取提高连对数

3. 注意声调的区别
4. 记住常见错词

- 查看"学习统计"了解学习情况
- 关注正确率和学习时长
- 鼓励孩子提高连对数

#### 重点复习
- 定期查看"最易错TOP5"
- 引导孩子使用错词复习功能
- 帮助孩子理解错词的含义

#### 激励方式
- 设定每日学习目标（如完成2关）
- 鼓励提高正确率（目标85%+）

## ❓ 常见问题

### Q1: 如何切换难度？
A: 每次开始新游戏时，在难度选择页重新选择即可。

### Q2: 错词本的词会自动消失吗？
A: 不会。只有以下两种情况会移除错词：
   1. 手动标记为"已掌握"
   2. 错词复习中连续答对3次自动标记

### Q3: 数据会丢失吗？
A: 数据存储在浏览器LocalStorage中，只有以下情况会丢失：
   1. 清除浏览器缓存
   2. 手动清除数据
   3. 更换浏览器或设备
   建议定期导出数据备份！

### Q4: 可以在手机上使用吗？
A: 应用支持响应式设计，但建议在平板或电脑上使用以获得最佳体验。

### Q5: 如何查看历史成绩？
A: 进入"学习统计"页面，查看"最近7天学习情况"部分。

### Q6: 可以跳过题目吗？
A: 不能跳过。每题必须作答才能进入下一题。

### Q7: 答错了怎么办？
A: 答错的词会自动记入错词本，可以稍后进行专项复习。

### Q8: 如何提高正确率？
A: 
   1. 多练习，熟能生巧
   2. 重点复习错词本
   3. 从简单难度开始，逐步提升
   4. 仔细读题，不要急于作答

## 🆘 遇到问题？

### 应用无法启动
1. 检查Node.js是否安装（运行 `node -v`）
2. 确认依赖已安装（运行 `npm install`）
3. 检查端口3000是否被占用
4. 查看命令行错误信息

### 页面显示异常
1. 刷新页面（F5）
2. 清除浏览器缓存
3. 使用Chrome或Edge浏览器
4. 检查浏览器控制台错误信息

### 数据异常
1. 进入设置页面
2. 清除所有数据
3. 重新开始使用
